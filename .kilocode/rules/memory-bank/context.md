## Design Specifications

### Color Palette
- **Primary Gradient**: Blue (#3b82f6) → Purple (#8b5cf6) → Pink (#ec4899)
- **Background**: Gradient from slate-50 → blue-50 → purple-50
- **Glass Effects**: White/70% with backdrop blur
- **Text**: Slate-800 (headings), Slate-600 (body)

### Typography
- **Headings**: Bold, gradient text for emphasis
- **Body**: Relaxed leading, slate colors
- **Font Features**: Ligatures enabled (rlig, calt)

### Animations
- **Float**: 6s ease-in-out infinite (±20px vertical)
- **Glow**: 2s ease-in-out alternate (shadow intensity)
- **Slide-up**: 0.5s ease-out (30px → 0)
- **Fade-in**: 0.6s ease-out (opacity 0 → 1)
- **Hover-lift**: -8px translate with shadow increase

## Technology Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript (strict mode)
- **Styling**: Tailwind CSS 4 with custom utilities
- **Icons**: Lucide React
- **AI**: Hugging Face Inference API
- **Search**: DuckDuckGo API
- **Package Manager**: Bun
